## 述要

此乃基于Qt框架之开发，囊括诸般功能模组，涉猎现代图形界面程序之要义。

## 纲领

是课业含天时察验、日课录存、忆事存记、设制管理、温变图示、智械对话等模组。

## 臻善之要

· 根基之能已具，然精进拓展之途犹存
· 次者可饰界面、增交互之巧、优效验等
